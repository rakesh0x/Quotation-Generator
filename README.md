# Manual Installation Guide

## 1. Clone the Repository

Repository:
[https://github.com/rakesh0x/Quotation-Generator](https://github.com/rakesh0x/Quotation-Generator)

```bash
git clone https://github.com/rakesh0x/Quotation-Generator.git
cd Quotation-Generator
```

---

## 2. Install Dependencies

Using Bun:

```bash
bun install
```

Or using npm:

```bash
npm install
```

---

## 3. Start the Development Server

Using npm:

```bash
npm run dev
```

Or using Bun:

```bash
bun run dev
```

---

## 4. Open the Application

Once the development server is running, open the following URL in your browser:

```bash
http://localhost:3000
```

You should now see the application running locally.
